## Hướng dẫn chạy project

1. Clone project:
git clone https://github.com/pngthanh/PTGear1.git

2. Copy vào htdocs (XAMPP)

3. Import database:
- Mở phpMyAdmin
- Tạo database: ptgear
- Import file ptgear.sql

4. Chạy:
http://localhost/ptgear1/index.php
